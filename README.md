# F Lab - Static Multipage Website

A static multipage website built with HTML, CSS, JavaScript, Tailwind CSS, and DaisyUI.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Build CSS for production:
```bash
npm run build
```

3. For development with watch mode:
```bash
npm run dev
```

## Project Structure

```
web/
├── assets/          # Static assets (images, logos, etc.)
├── dist/           # Compiled CSS output (generated)
├── js/             # JavaScript files
├── pages/          # Additional HTML pages
├── index.html      # Home page
├── styles.css      # Tailwind CSS source file
├── tailwind.config.js  # Tailwind configuration
└── package.json    # Dependencies and scripts
```

## Features

- ✅ Tailwind CSS (latest version)

- ✅ Custom color palette integrated
