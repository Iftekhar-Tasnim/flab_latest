# F Lab Website - Development Plan

## Site Overview
This is a static multipage website for F Lab organization. The site will showcase the organization's work, team, programs, publications, and more.

## Technology Stack
- HTML5
- CSS (Tailwind CSS v3.4.1)
- JavaScript (Vanilla JS)
- DaisyUI v4.12.10 (UI Components)
- Official Color Palette:
  - Yale Blue (Primary)
  - Platinum (Secondary)
  - Fresh Sky (Accent)
  - Prussian Blue (Dark/Text)

## Site Structure
1. **Home** - Landing page
2. **About** - Organization information
3. **Programmes** - Programs/initiatives offered
4. **Publications** - Research papers, articles, reports
5. **Team** - Team members and staff

7. **Blog** - Blog posts and news
8. **Shop** - Products/merchandise
9. **Contact Us** - Contact information and form

## Design System

### Typography
- Headings: Bold, Prussian Blue
- Body text: Platinum shades
- Links: Yale Blue with hover effects

### Components
- Buttons: Yale Blue background, white text
- Cards: White background, Platinum borders
- Navigation: White background, active state in Yale Blue

### Spacing
- Consistent padding and margins
- Responsive spacing for mobile/tablet/desktop

### Images
- Optimized file sizes
- Consistent aspect ratios
- Alt text for accessibility

## Development Workflow
1. Plan page structure and content
2. Design layout and components
3. Implement HTML structure
4. Apply styling with Tailwind CSS
5. Add interactivity with JavaScript
6. Test responsiveness
7. Review and refine
8. Move to next page
